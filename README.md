📐 Java Design Patterns Repository  
Welcome to the Java Design Patterns Practice Repository!

This repo contains a curated set of the most important and widely used **Design Patterns**, implemented in clean and easy-to-understand **Java** code.

Perfect for:

🔍 System design interviews  
👨‍💻 Writing scalable and maintainable code  
🧠 Mastering OOP principles through real-world examples  

---

✨ What You'll Find Here

✅ Clean, modular Java code for each pattern  
📂 Organized by category (Creational, Structural, Behavioral, FRP)  
📝 Well-commented examples with real-world analogies  
🚀 Easy to run with `main()` methods in each pattern  

---

📁 Categories Covered

### 🔨 Creational Patterns
- Singleton
- Factory
- Abstract Factory
- Builder
- Prototype

### 🧱 Structural Patterns
- Adapter
- Decorator
- Facade
- Proxy
- Composite
- Bridge

### 🔄 Behavioral Patterns
- Strategy
- Observer
- Command
- State
- Mediator
- Chain of Responsibility
- Template Method
- Visitor

### ⚡ Reactive / FRP (Functional Reactive Programming)
- Observer in reactive style
- Functional pipeline examples (Rx-like)
- Event-driven design
